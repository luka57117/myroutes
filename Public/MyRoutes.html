My Routes was created as an sample add-on, and is meant for demonstration purposes only. It allows to compare map and routing services from different providers on the same page.
The number of map engines requests this add-on produces is limited by the available quota per API KEY.

Short induction:

Select menu Add-ons - My Routes - Show Sidebar to call add-on's sidebar.
Press this button to use selected in your document addresses as your waypoints.
Press this button to add selected in your document addresses to waypoints list.
Drag and drop waypoints in the list to change the order.
Double click the waypoint in the list to remove it.
Show - press button to open the add-on window with your routes.
Browse - press button to open new browser tab with your routes.
Double click the bottom branding text to toggle interface translation on/off.
Select menu Add-ons - My Routes - Settings or
Double click the logo to set your own map API KEYs and customize interface translation.
